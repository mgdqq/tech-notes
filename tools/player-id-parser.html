PlayerId Parser

使用 BigInt 解析 PlayerId，展示 timestamp、generatorId、sequence 以及 generatorId 内部的 loginNodeId 与 bornChannelId。

Bit Layout

Timestamp 29 bits
timestampSeconds
GeneratorId 20 bits
generatorId
LoginNodeId high 6 bits
loginNodeId
BornChannelId low 14 bits
bornChannelId
Sequence 14 bits
sequence
timestampSeconds
generatorId
loginNodeId

sequence
sequence = playerId & ((1 << 14) - 1)
generatorId = (playerId >> 14) & ((1 << 20) - 1)
timestampSeconds = playerId >> (14 + 20)
loginNodeId = generatorId >> 14
bornChannelId = generatorId & ((1 << 14) - 1)
